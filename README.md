# Space-tourism

A visually engaging, UI/UX-focused React application styled with Tailwind CSS. Includes interactive pages for destinations, crew, and technology. Deployed via Netlify.

## 🛠️ Technologies Used  

- **React**
- **Tailwind CSS**
- **Netlify**

## 🌐 Live Demo 
https://demospacetourism.netlify.app/
